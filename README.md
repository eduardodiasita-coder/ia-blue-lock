# ⚽ Blue Lock Ego Chat

Uma inteligência artificial especialista e fascinada pelo universo do anime e mangá **Blue Lock**. Desenvolvida para responder com a personalidade empolgada e egoísta dos personagens da obra.

## 🚀 Tecnologias Utilizadas
* **Python** (Linguagem base)
* **Google Gemini API** (Modelo `gemini-3.5-flash` para o cérebro da IA)
* **Streamlit** (Para a criação da interface web visual)
* **RAG (Geração Aumentada por Recuperação)** (Conceito usado para alimentar a IA com arquivos de texto locais)

## 🧠 Como funciona?
O projeto utiliza um arquivo de contexto (`bluelock.txt`) contendo informações detalhadas sobre as armas e personalidades dos jogadores (Isagi, Bachira, Rin, etc.). A IA consulta essa base de dados antes de formular as respostas, garantindo precisão nos fatos e mantendo o tom correto da obra.
